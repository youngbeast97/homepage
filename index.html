Z czego składa się frontend?

Frontend składa się z 3 podstawowych elementów:

  • HTML
  • JavaScript
  • CSS
HTML (ang. HyperText Markup Language) - jest to wg. Wikipedii język znaczników stosowany do tworzenia dokumentów hipertekstowych.
Nie jest to wbrew pozorom język programowania, a za takiego jest uważany przez wielu ludzi rozpoczynających swoją przygodę z Frontendem.

Kolejnym elementem niezbędnym do nauki we frontendzie jest CSS czyli kaskadowe arkusze stylów czyli język służący do opisu formy prezentacji elementów zapisanych w HTML,
co w skrócie oznacza, że stosując CSS dbamy o formę podania treści zapisanej w HTMLu. Żródłem wiedzy dla HTMla i CSS jest organizacja W3C, która dba o poprawny standard tworzenia stron WWW, chociaż ich ustalenia i publikacje nie mają mocy prawnej.
